Learn more on HTML

I don't have much experience when it comes to coding or development, but here are some links I think are pretty helpful

  • <details>

    • The HTML element creates a disclosure widget in which information is visible only when the widget is toggled into an "open" state. Learn more from MWD.

  • <div>

    • The HTML element is the generic container for flow content. It has no effect on the content or layout until styled in some way using CSS. Learn more at MWD.

  • <progress>

    • The HTML element displays an indicator showing the completion progress of a task, typically displayed as a progress bar. I chose this one because it looked pretty cool when reading on it. Learn more on MWD.
